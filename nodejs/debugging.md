# Debugging

- [Debugging Guide](https://nodejs.org/en/docs/guides/debugging-getting-started/)

## Chrome DevTools

1. Run application: `node --inspect index.js`.
2. Open in chromium: `chrome://inspect`.
3. Click the `Open dedicated DevTools for Node` link.
